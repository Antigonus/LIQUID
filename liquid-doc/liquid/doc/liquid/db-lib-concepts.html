2.2 db-lib

db-lib is built on top of Ryan Culpepper’s DB lib.

The liquid/db-lib library provides an interface for specific purpose of providing persistent storage of LISP data in tabular format in a database.

db-lib was written to support the dataplex library, though it may be used by anyone.
